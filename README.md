# DayMade Expo App

## Run Locally
1. Install Expo Go on your phone (iOS App Store / Google Play).
2. Upload this repo to GitHub.
3. Connect GitHub to [Expo EAS](https://expo.dev) and build.
4. Scan the QR code in Expo Go to open the app.

## Features
- Login / Signup
- Home Dashboard
- Memberships
- Tasks
- Concierge Chat
- Marketplace
- Operator Panel
